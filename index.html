Box Model

El modelo de caja es fundamental en CSS. Define cómo se estructura cada elemento en la página, considerando el contenido, padding, borde y margen.

Margin
Border
Padding
Content

width

Define el ancho del contenido de un elemento (sin incluir padding, borde o margen).

Valores posibles: longitud (px, em, rem), porcentaje (%), auto, max-content, min-content, fit-content

.elemento { width: 200px; }

Consejo: Cuando se usa box-sizing: border-box, el width incluye el padding y el borde.

height

Define la altura del contenido de un elemento (sin incluir padding, borde o margen).

Valores posibles: longitud (px, em, rem), porcentaje (%), auto, max-content, min-content, fit-content

.elemento { height: 100px; }

Consejo: Es mejor dejar que los elementos determinen su altura naturalmente cuando sea posible.

min-width / max-width

Establecen el ancho mínimo y máximo de un elemento, útil para diseño responsive.

Valores posibles: longitud (px, em, rem), porcentaje (%), none (para max-width)

.elemento { min-width: 100px; max-width: 300px; }

Caso de uso: Limitar el ancho de un párrafo para mantener la legibilidad en pantallas grandes.

min-height / max-height

Establecen la altura mínima y máxima de un elemento.

Valores posibles: longitud (px, em, rem), porcentaje (%), none (para max-height)

.elemento { min-height: 50px; max-height: 200px; }

Caso de uso: Crear áreas de contenido con altura fija pero que puedan crecer hasta cierto punto.

margin

Define el espacio exterior alrededor de un elemento, creando distancia entre este y otros elementos.

Valores posibles: longitud (px, em, rem), porcentaje (%), auto

/* Todos los lados */ .elemento { margin: 10px; } /* Vertical | Horizontal */ .elemento { margin: 10px 20px; } /* Superior | Horizontal | Inferior */ .elemento { margin: 10px 20px 15px; } /* Superior | Derecha | Inferior | Izquierda */ .elemento { margin: 10px 20px 15px 25px; }

Consejo: Los márgenes verticales adyacentes se colapsan en CSS, tomando el valor del margen más grande.

padding

Define el espacio interior entre el contenido de un elemento y su borde.

Valores posibles: longitud (px, em, rem), porcentaje (%)

/* Todos los lados */ .elemento { padding: 10px; } /* Vertical | Horizontal */ .elemento { padding: 10px 20px; } /* Superior | Horizontal | Inferior */ .elemento { padding: 10px 20px 15px; } /* Superior | Derecha | Inferior | Izquierda */ .elemento { padding: 10px 20px 15px 25px; }
Contenido

Consejo: A diferencia de los márgenes, el padding no se colapsa y siempre suma al tamaño del elemento (a menos que uses box-sizing: border-box).

box-sizing

Determina cómo se calcula el ancho y alto total de un elemento.

Valores posibles: content-box, border-box

/* El ancho/alto solo incluye el contenido */ .elemento { box-sizing: content-box; } /* El ancho/alto incluye contenido, padding y borde */ .elemento { box-sizing: border-box; }
content-box
border-box

Práctica recomendada: Muchos desarrolladores aplican box-sizing: border-box a todos los elementos para facilitar el cálculo de dimensiones.

Background

Las propiedades de fondo permiten controlar el color, imagen y comportamiento del fondo de los elementos.

background-color

Define el color de fondo de un elemento.

Valores posibles: nombre de color, código hexadecimal, rgb(), rgba(), hsl(), hsla(), transparent

/* Nombre de color */ .elemento { background-color: blue; } /* Hexadecimal */ .elemento { background-color: #3498db; } /* RGB con transparencia */ .elemento { background-color: rgba(52, 152, 219, 0.5); } /* HSL (Hue, Saturation, Lightness) */ .elemento { background-color: hsl(204, 70%, 53%); }

Consejo: Usa rgba() o hsla() cuando necesites transparencia en el color de fondo.

background-image

Establece una o más imágenes como fondo de un elemento.

Valores posibles: url(), gradientes (linear-gradient, radial-gradient), none

/* Imagen de fondo */ .elemento { background-image: url('imagen.jpg'); } /* Gradiente lineal */ .elemento { background-image: linear-gradient(to right, #3498db, #2c3e50); } /* Gradiente radial */ .elemento { background-image: radial-gradient(circle, #3498db, #2c3e50); } /* Múltiples fondos */ .elemento { background-image: url('patrón.png'), linear-gradient(to right, #3498db, #2c3e50); }

Consejo: Puedes combinar múltiples imágenes y gradientes separándolos con comas. Se mostrarán en el orden listado (el primero arriba).

background-size

Controla el tamaño de la imagen de fondo.

Valores posibles: auto, longitud (px, em, rem), porcentaje (%), cover, contain

/* Tamaño específico */ .elemento { background-size: 200px 100px; } /* Porcentaje relativo al contenedor */ .elemento { background-size: 50% 50%; } /* Cubrir todo el contenedor (puede recortar) */ .elemento { background-size: cover; } /* Contener la imagen completa (puede dejar espacios) */ .elemento { background-size: contain; }
auto
cover
contain

Caso de uso: background-size: cover es ideal para imágenes de fondo que deben llenar completamente un elemento, como cabeceras o héroes.

background-position

Define la posición inicial de la imagen de fondo.

Valores posibles: palabras clave (top, right, bottom, left, center), longitud (px, em, rem), porcentaje (%)

/* Palabras clave */ .elemento { background-position: top right; } /* Porcentajes */ .elemento { background-position: 25% 75%; } /* Valores específicos */ .elemento { background-position: 10px 20px; }

Consejo: Los porcentajes se refieren tanto al tamaño del elemento como al de la imagen. 0% 0% es la esquina superior izquierda, 100% 100% es la esquina inferior derecha.

background-repeat

Controla cómo se repite la imagen de fondo.

Valores posibles: repeat, repeat-x, repeat-y, no-repeat, space, round

/* Repetir en ambas direcciones */ .elemento { background-repeat: repeat; } /* Repetir solo horizontalmente */ .elemento { background-repeat: repeat-x; } /* Repetir solo verticalmente */ .elemento { background-repeat: repeat-y; } /* Sin repetición */ .elemento { background-repeat: no-repeat; } /* Espaciar las repeticiones uniformemente */ .elemento { background-repeat: space; } /* Ajustar las repeticiones para evitar recortes */ .elemento { background-repeat: round; }
repeat
repeat-x
no-repeat

background (propiedad abreviada)

Permite establecer todas las propiedades de fondo en una sola declaración.

/* Orden: color imagen posición/tamaño repeat attachment */ .elemento { background: #3498db url('imagen.jpg') center/cover no-repeat fixed; }

Consejo: No es necesario incluir todas las propiedades en la abreviatura. Las omitidas tomarán sus valores por defecto.

Fonts

Las propiedades de fuente permiten controlar la apariencia del texto, incluyendo el tipo de letra, tamaño, peso y estilo.

font-family

Define la familia de fuentes que se utilizará para mostrar el texto.

Valores posibles: nombre de fuente específica, familia genérica (serif, sans-serif, monospace, cursive, fantasy), lista de fuentes

/* Fuente específica con alternativas */ .elemento { font-family: 'Roboto', Arial, sans-serif; } /* Familia genérica */ .elemento { font-family: sans-serif; } /* Fuente con espacios en el nombre */ .elemento { font-family: 'Times New Roman', Times, serif; }

Texto con serif

Texto con sans-serif

Texto con monospace

Texto con cursive

Consejo: Siempre incluye una familia genérica al final de la lista como respaldo en caso de que las fuentes específicas no estén disponibles.

font-size

Define el tamaño del texto.

Valores posibles: longitud (px, em, rem, pt), porcentaje (%), palabras clave (small, medium, large, etc.)

/* Tamaño absoluto en píxeles */ .elemento { font-size: 16px; } /* Tamaño relativo al elemento padre */ .elemento { font-size: 1.5em; } /* Tamaño relativo al elemento raíz (html) */ .elemento { font-size: 1.2rem; } /* Tamaño en porcentaje */ .elemento { font-size: 120%; } /* Palabras clave */ .elemento { font-size: large; }

Texto pequeño (12px)

Texto mediano (16px)

Texto grande (24px)

Texto extra grande (32px)

Práctica recomendada: Usa unidades relativas como rem para mejorar la accesibilidad y permitir que el texto se ajuste a las preferencias del usuario.

font-weight

Define el grosor o peso de la fuente.

Valores posibles: normal, bold, bolder, lighter, 100-900

/* Valores predefinidos */ .elemento { font-weight: normal; } /* Equivale a 400 */ .elemento { font-weight: bold; } /* Equivale a 700 */ /* Valores numéricos */ .elemento { font-weight: 300; } /* Light */ .elemento { font-weight: 400; } /* Regular */ .elemento { font-weight: 500; } /* Medium */ .elemento { font-weight: 700; } /* Bold */ .elemento { font-weight: 900; } /* Black */

Peso 300 (Light)

Peso 400 (Regular)

Peso 500 (Medium)

Peso 700 (Bold)

Peso 900 (Black)

Nota: No todas las fuentes tienen todos los pesos disponibles. Si se especifica un peso que no existe, el navegador usará el peso más cercano disponible.

font-style

Define el estilo de la fuente.

Valores posibles: normal, italic, oblique

/* Estilo normal */ .elemento { font-style: normal; } /* Estilo itálico */ .elemento { font-style: italic; } /* Estilo oblicuo */ .elemento { font-style: oblique; }

Texto con estilo normal

Texto con estilo italic

Texto con estilo oblique

Diferencia: italic es una versión diseñada específicamente de la fuente, mientras que oblique es una versión inclinada artificialmente de la fuente normal.

line-height

Define la altura de línea o interlineado.

Valores posibles: normal, número (sin unidad), longitud (px, em, rem), porcentaje (%)

/* Valor predeterminado */ .elemento { line-height: normal; } /* Número sin unidad (recomendado) */ .elemento { line-height: 1.5; } /* Longitud específica */ .elemento { line-height: 24px; } /* Porcentaje */ .elemento { line-height: 150%; }

Texto con line-height: 1
Segunda línea de texto para mostrar el interlineado

Texto con line-height: 1.5
Segunda línea de texto para mostrar el interlineado

Texto con line-height: 2
Segunda línea de texto para mostrar el interlineado

Práctica recomendada: Usa un número sin unidad para line-height, ya que este valor se hereda proporcionalmente, lo que facilita mantener una jerarquía tipográfica coherente.

font (propiedad abreviada)

Permite establecer varias propiedades de fuente en una sola declaración.

Orden: font-style font-variant font-weight font-size/line-height font-family

/* Sintaxis completa */ .elemento { font: italic bold 16px/1.5 'Roboto', sans-serif; } /* Sintaxis mínima (solo tamaño y familia) */ .elemento { font: 16px 'Roboto', sans-serif; }

Texto con font: italic bold 18px/1.6 'Roboto', sans-serif

Nota: Al usar la propiedad abreviada font, los valores omitidos se restablecen a sus valores iniciales. Solo font-size y font-family son obligatorios.

Propiedades Misceláneas

Estas propiedades controlan diversos aspectos de la presentación que no encajan en otras categorías, como la visibilidad, el cursor, o el comportamiento de desbordamiento.

opacity

Controla la transparencia de un elemento.

Valores posibles: número entre 0 y 1

/* Completamente opaco (predeterminado) */ .elemento { opacity: 1; } /* Semitransparente */ .elemento { opacity: 0.5; } /* Completamente transparente (invisible) */ .elemento { opacity: 0; }
1.0
0.7
0.3

Nota: La propiedad opacity afecta a todo el elemento y sus descendientes. Para aplicar transparencia solo al color de fondo, usa rgba() o hsla().

cursor

Define el tipo de cursor que se muestra cuando el puntero está sobre un elemento.

Valores posibles: auto, default, pointer, text, move, not-allowed, grab, zoom-in, etc.

/* Cursor predeterminado */ .elemento { cursor: default; } /* Cursor de mano (para elementos clickeables) */ .elemento { cursor: pointer; } /* Cursor de texto */ .elemento { cursor: text; } /* Cursor de prohibido */ .elemento { cursor: not-allowed; } /* Cursor personalizado */ .elemento { cursor: url('mi-cursor.png'), auto; }
default
pointer
text
not-allowed

Caso de uso: Usa cursor: pointer para elementos que realizan una acción al hacer clic, como botones o enlaces.

Efectos Visuales

CSS3 introdujo varias propiedades para crear efectos visuales avanzados sin necesidad de imágenes o JavaScript.

box-shadow

Añade una o más sombras a un elemento.

Valores: desplazamiento-x desplazamiento-y difuminado propagación color inset

Diseño Responsive

El diseño responsive permite que los sitios web se adapten a diferentes tamaños de pantalla y dispositivos, proporcionando una experiencia óptima para todos los usuarios.

@media

Regla at que permite aplicar estilos basados en características del dispositivo o del navegador.

Características comunes: width, height, orientation, aspect-ratio, resolution

/* Estilos para pantallas pequeñas */ @media (max-width: 600px) { .elemento { width: 100%; font-size: 14px; } } /* Estilos para pantallas medianas */ @media (min-width: 601px) and (max-width: 1024px) { .elemento { width: 50%; font-size: 16px; } } /* Estilos para orientación horizontal */ @media (orientation: landscape) { .elemento { display: flex; } } /* Estilos para dispositivos de alta resolución */ @media (min-resolution: 2dppx) { .imagen { background-image: url('imagen@2x.jpg'); } }

Redimensiona la ventana para ver cómo cambia este elemento

Enfoque: Es recomendable utilizar un enfoque "mobile-first", donde se diseña primero para dispositivos móviles y luego se añaden estilos para pantallas más grandes con @media (min-width: ...).

Animaciones

Las animaciones CSS permiten crear transiciones y cambios de estado sin necesidad de JavaScript.

@keyframes

Define los estados de una animación en diferentes puntos del tiempo.

/* Definición básica con from/to */ @keyframes slide { from { transform: translateX(0); } to { transform: translateX(100px); } } /* Definición con porcentajes */ @keyframes bounce { 0% { transform: translateY(0); } 50% { transform: translateY(-20px); } 100% { transform: translateY(0); } } /* Múltiples propiedades */ @keyframes fade-move { 0% { opacity: 0; transform: translateY(20px); } 100% { opacity: 1; transform: translateY(0); } }

Consejo: Puedes animar casi cualquier propiedad CSS, pero las más eficientes son transform y opacity.

animation

Propiedad abreviada para aplicar animaciones a un elemento.

Componentes: animation-name, animation-duration, animation-timing-function, animation-delay, animation-iteration-count, animation-direction, animation-fill-mode, animation-play-state

/* Sintaxis básica */ .elemento { animation: nombre duración timing-function delay iteraciones dirección fill-mode play-state; } /* Ejemplo simple */ .elemento { animation: slide 2s ease 0s infinite normal forwards running; } /* Ejemplo abreviado */ .elemento { animation: slide 2s ease infinite; } /* Múltiples animaciones */ .elemento { animation: slide 2s ease, fade 3s linear; }

animation-timing-function

Define cómo progresa la animación a lo largo de su duración.

Valores posibles: ease, linear, ease-in, ease-out, ease-in-out, cubic-bezier(), steps()

/* Aceleración y desaceleración suave (predeterminado) */ .elemento { animation-timing-function: ease; } /* Velocidad constante */ .elemento { animation-timing-function: linear; } /* Aceleración inicial */ .elemento { animation-timing-function: ease-in; } /* Desaceleración final */ .elemento { animation-timing-function: ease-out; } /* Aceleración inicial y desaceleración final */ .elemento { animation-timing-function: ease-in-out; } /* Curva personalizada */ .elemento { animation-timing-function: cubic-bezier(0.42, 0, 0.58, 1); } /* Animación por pasos */ .elemento { animation-timing-function: steps(5, end); }
